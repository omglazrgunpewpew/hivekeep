Self-hosted & self-improving

Une base qui grandit avec vous.

Un seul conteneur, zéro infra externe — et des agents qui étendent eux-mêmes la plateforme.

📦 Un seul docker run

Pas de Postgres, Redis, Mongo ni broker de queue. Bun + SQLite, un binaire, vos données chez vous.

$ docker run -d -p 3000:3000 \
  -v hivekeep-data:/app/data ghcr.io/.../hivekeep
# puis ouvrez le navigateur — Queenie s'occupe du reste

🛠️ Self-improving

Vos Agents écrivent leurs propres outils, construisent des mini-apps intégrées et publient des plugins.

custom toolsmini-appsplugins NPMMCP

🔒 Vos secrets ne voient jamais le LLM

Coffre chiffré AES-256-GCM. Les Agents référencent une clé du vault — sans jamais en lire la valeur.
